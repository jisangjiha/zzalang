## 프로젝트 소개
자유롭게 글을 작성할 수 있는 게시판 서비스입니다.

## 팀원 구성
- 홍석민: BackEnd
- 홍지상: FrontEnd

## FrontEnd 개발 환경
- Core: React, TypeScript, Html5/CSS3
- Styling: CSS Modules
- UI Feedback: React Toastify
- Build Tool: Vite
- Package Manager: pnpm
- Routing: React Router DOM

## 주요 기능
- 게시판 기본 기능
  - 회원가입 / 로그인 기반 게시글 작성, 조회 기능
  - 게시글 목록 및 상세 페이지 제공
- 카테고리 기반 게시글 분류
  - 게시글 작성 시 카테고리 선택
  - 카테고리별 게시글 필터링 제공
- 검색 기능
  - 제목 + 내용 기준 게시글 검색
  - 검색 조건 유지한 상태에서 목록 탐색 가능
- 페이지네이션
  - 게시글 목록 페이지네이션 처리
  - 페이지 이동 시 불필요한 리렌더링 최소화
